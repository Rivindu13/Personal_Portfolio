Projects Overview

Here is a more detailed look at some of the projects I have worked on. Each project helped me practise different front-end skills such as layout, styling, and interactivity.

Technologies: HTML, CSS Grid/Flexbox, JavaScript

Product Landing Page

A simple landing page for a fictional product, focusing on clear call to action buttons, information sections, and responsive typography.

Key features: responsive columns, button styles, clean typography.

Technologies: HTML, CSS (Flexbox)

JavaScript Mini App

A small interactive app (for example, a calculator or to-do list) that allowed me to practise event handling and updating the DOM based on user input.

Key features: input validation, dynamic DOM updates.

Technologies: HTML, CSS, JavaScript
